## Convention

Writing here is *synthetic*, not reactive. Documents in this directory pull threads across multiple sources, propose new framings, or track how the investigation's thinking is evolving over time.

New items land here when:

- A thread emerges across reading that warrants its own treatment
- A tension or question needs articulation independent of any single proposal version
- An essay is completed or received feedback worth retaining

Keep iterations visible. If a document is substantially revised, the older version can be archived alongside rather than overwritten, when preserving the trajectory is useful.

## What belongs here

- Synthesis essays that span multiple primary sources
- Tension trackers, conceptual maps, and cross-cutting working documents
- Peer-reviewed or feedback-marked drafts retained as context

## What doesn't belong here

- Per-paper reading notes (those go in `../reading/notes/`)
- Proposal versions and their scaffolds (those go in `../proposals/`)
- Bibliography entries (those go in `../reading/reading-log.md`)
