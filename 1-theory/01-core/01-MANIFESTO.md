# Manifesto

Coordination theory is incomplete. This work exists to demonstrate that, and to explore what's missing.

## The Conviction

The models we use to organise collective action between individuals, within institutions and across nations have not been seriously revised in centuries. They are comfortable rather than correct. Challenging them requires admitting that the foundations we've built everything on are not adequate for where we are now and for what's coming. And the people with the power to change them benefit most from leaving them exactly as they are.

These models assume cooperation requires either shared beliefs or aligned self-interest. That you and I must either agree on a worldview or share an immediate interest before we can work together on anything that matters. Shared beliefs and aligned interests do make coordination easier. Nobody disputes that. The problem is treating them as prerequisites, because when they're unattainable, these models offer nothing. No playbook, no fallback, no path forward. Just gridlock dressed up as prudence.

Convergence is precisely what is often unavailable. Families coordinate across profound difference every single day. Communities hold together despite disagreements that would paralyse a parliament. Open-source developers who've never met build cathedrals of code without sharing ideology, nationality or motivation. The UK, USA and Australia coordinate on catastrophic security risks through arrangements like AUKUS without sharing strategic doctrines, democratic constraints or the same definition of what regional security requires. The evidence that coordination across genuine difference is possible and ordinary is everywhere. What we lack is not the capacity. It's the theory, the language and the institutional vocabulary to describe what makes it work.

## The Question

If value convergence is neither sufficient nor necessary for coordination, what actually grounds it?

Game theory assumes you can build a common payoff matrix. When agents hold genuinely incommensurable values, that assumption fails by design. The circumstances of agents whose values don't resolve into a shared ranking have mostly been treated as a management problem to be navigated pragmatically, rather than a formal condition worth theorising. This work takes it as the formal condition.

The hypothesis is that structural features of interdependence (shared risk topology, shared constraints, network entanglement, procedural commitments, possibly others) can ground coordination where value convergence cannot. Whether this hypothesis holds, and under what formal conditions, is what the investigation is trying to establish.

## The Work

This is a public research log, currently grounded in doctoral-track philosophical work. Where convergence is achievable, the framework welcomes it. Where it isn't, the investigation asks what else might do the work convergence was expected to do.

Like a compass, the intended framework provides shared orientation without dictating the route. Different actors, with genuinely different values and genuinely different goals, navigating by the same reference points toward outcomes none of them could achieve alone. Whether a defensible compass of this kind can actually be built is what the work is testing.

This is early-stage work. It will be wrong in many places. It will be incomplete for years. It is being built in the open because the kind of coordination it describes can only be developed through the kind of coordination it describes: people with different perspectives, different expertise and different convictions contributing to something larger than any single viewpoint.

## The Stakes

If the hypothesis is right, the implications extend well beyond any single domain. The current investigation narrows to one philosophical question and one applied case study. If the formal account holds there, it should generalise to other coordination problems where agents share structural interdependence but not values.

If the hypothesis is wrong, we need to know that too, clearly and honestly, so the effort can be redirected toward what actually works before the window closes.

Start small. Aim big.
