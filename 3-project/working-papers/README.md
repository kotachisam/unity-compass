# Working Papers

Work-in-progress documents supporting the investigation. Unpolished, iterative, built to evolve.

Each subdirectory below has its own README covering contents and conventions. On GitHub, clicking into any of them renders that README automatically.

## Contents

**[proposals/](proposals/)** ([README](proposals/README.md)): doctoral research proposal iterations.
- `research-proposal-v1.md`: original Oxford DPhil proposal (manifesto-adjacent framing, Ruth Chang targeted)
- `research-proposal-v2.md`: current LSE MPhil/PhD proposal (narrowed to structural hypothesis, Alexander + Bradley targeted)
- `rpv2-plan.md`: scaffold and pitfall notes for the v2 rewrite

**[reading/](reading/)** ([README](reading/README.md)): primary-source research log.
- `reading-log.md`: bibliography with priority, status and access notes
- `notes/` ([README](reading/notes/README.md)): per-paper reading notes, one file per source

**[writing/](writing/)** ([README](writing/README.md)): essays and cross-cutting working documents.
- `philosophical-tensions.md`: tensions between the broader Unity Compass repository and the narrower v2 thesis. Not inconsistencies to be resolved editorially, but open questions the investigation has to address.
- `on-logos.pdf`: foundational essay on the narrowing of logos from Heraclitus to formal logic, with peer feedback from Dr Michaelis Michael (UNSW). Referenced throughout the core documents.

## Status

Working material, not publication-ready. Everything here is either being actively developed or retained as a historical artefact (e.g. v1 proposal). Revisions should leave older versions in place where they track the evolution of thinking.
