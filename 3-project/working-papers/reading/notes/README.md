# Reading Notes

Per-paper notes captured during primary source reading.

## Contents

No note files yet. As papers get read, each one should produce a note file here.

## Convention

One file per paper or book. Filename follows the pattern `author-year-short-title.md` (e.g. `alexander-2015-cheap-talk.md`, `bradley-2022-social-ethics-ambiguity.md`).

Each note should capture, at minimum:

- **Citation** (full)
- **Core claim** (one or two sentences)
- **Methodology** (formal, empirical, conceptual)
- **Key concepts introduced**
- **Where it bites on the investigation** (what question of the thesis this paper informs, extends, or contradicts)
- **Quotes worth retaining** (with page numbers)
- **Open questions** raised for the investigation

Notes are honest working documents, not polished summaries. Uneven depth is fine. Rough reactions, points of disagreement, hunches worth checking: all belong here.

## Relationship to the Reading Log

The [reading-log](../reading-log.md) tracks what exists to be read and what's been read. This directory holds the substance of what was extracted from each reading. When a paper moves from `reading` to `read` status in the log, a note file here should exist, and the log entry should link to it.
