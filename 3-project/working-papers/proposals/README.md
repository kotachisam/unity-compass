# Proposals

Doctoral research proposal iterations. Each major revision kept as a discrete file so the evolution of thinking stays legible.

## Contents

**[research-proposal-v1.md](research-proposal-v1.md)**: original DPhil proposal (Oxford, Philosophy). Written around Ruth Chang's work on hard choices and voluntarist agency, framed as "post-ideological coordination for existential governance" across multiple domains. Historical artefact. Retained unchanged as the starting point the investigation has since narrowed from.

**[research-proposal-v2.md](research-proposal-v2.md)**: current MPhil/PhD proposal (LSE, Philosophy). Narrowed to the formal question of whether structural features of interdependence can ground coordination among agents with incommensurable values. Targets supervision by J. McKenzie Alexander and Richard Bradley. Application window: May 2026 for September 2026 entry.

**[rpv2-plan.md](rpv2-plan.md)**: scaffold and checklist used to rebuild v2 from v1. Includes pitfall notes from the critical analysis of v1 (AI-voice tells, defensive comprehensiveness, domain sprawl). Useful for subsequent revisions as a structural reference.

## Convention

New proposal iterations land here as `research-proposal-vN.md`. Keep prior versions intact rather than overwriting, so the trajectory stays visible. Supporting scaffolds and plan documents sit alongside the version they scaffolded.
